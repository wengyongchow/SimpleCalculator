1. This is a simple calculator consist of 4 arithmetic calculation(Addition, Substraction, Multiplication and Division).

2. When the program launched, it requirs two numbers as the input.

3. The range of the number must be in between -10000 to 10000, otherwise error message will prompt.

4. After getting the input, the program will direct display the 4 arithmetic calculation of the two numbers.

5. 2 decimal point will be used in division calucation. It can display the negative result as well as the positive result.
